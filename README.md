# Expense Tracker Bot

## 📌 Опис

Telegram-бот для обліку особистих витрат, створений за допомогою n8n.

## 🚀 Можливості

- Додавання витрат через Telegram
- Автоматичний запис у Google Sheets
- Перегляд історії витрат
- Підрахунок загальної суми

## 🛠 Використані технології

- n8n
- Telegram Bot API
- Google Sheets

## 🔄 Логіка роботи

1. Користувач надсилає повідомлення боту.
2. n8n обробляє повідомлення.
3. Дані записуються в Google Sheets.
4. Бот повертає відповідь користувачу.
